
# Global Conflict Market Dashboard

Tracks how geopolitical conflicts affect financial markets.

Assets monitored
- Gold
- Crude Oil
- Nifty 50
- S&P 500
- VIX

Features
- Daily market changes
- Weekly market changes
- Conflict risk signal
- Correlation heatmap
- Interactive charts

## Run locally

pip install -r requirements.txt
streamlit run app.py

## Deploy free

1. Create a GitHub repository
2. Upload these files
3. Go to https://share.streamlit.io
4. Deploy using app.py
